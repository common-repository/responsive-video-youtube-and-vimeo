== Description ==

Plugin Responsive Youtube And Vimeo, This wordpress plugin is simply used in your editor


== Installation ==

1.Install plugin from Wordpress menu Plugins > Add New
2.Activate the plugin
3.Well done.
4.Insert your shortcode to the post,page, Widget 

5.List Shorcode demo:
Shortcode Youtube 
[video-responsive v="?v=60ItHLz5WEA"]
 Shortcode vimeo
[video-responsive v="295861243"]

6. Place `<?php do_action('video-responsive'); ?>` in your templates
